##	Live Scheduler or Run Current Processes

![image](https://user-images.githubusercontent.com/92469329/235043206-752d688a-5eae-4346-bdc1-db566fa879ac.png)

 
You can choose between running processes as:

### 1.	A live scheduler is run with each 1 unit of time mapped to 1 second and it is shown in the top of GUI
Note : A new process can be added dynamically while the scheduler is running.
Note : Remaining burst time, waiting time, turnaround time table is updated as time progresses.

![image](https://user-images.githubusercontent.com/92469329/235043286-eab2e568-85b2-4d5d-b2af-58ff22d799ab.png)

![image](https://user-images.githubusercontent.com/92469329/235043300-06488e1c-86d6-4296-b8a4-795122528895.png)


 
 






### 2.	Run the currently existing processes and it shown below Live Scheduler
Note : Turnaround Time and Waiting Time for every process is updated on table after executing processes
Note : Processes is not removed after running in this option but only updates turnaround and waiting time for every process after executing all processes
